# GitHub Actions

These workflows automate the team workflow.

| Workflow | Purpose |
|----------|---------|
| [Deploy Documentation](./deploy-docs) | Automation workflow |
| [Doc Change Notification](./doc-change-notify) | Automation workflow |
| [Trigger n8n Tests](./trigger-n8n) | Automation workflow |

## How They Work

1. **Doc Change Notify** - Notifies team when documentation changes
2. **Trigger n8n** - Sends webhooks to n8n for test automation
