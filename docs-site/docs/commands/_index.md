# Claude Commands

These commands are available in Claude Code for this project.

| Command | Description |
|---------|-------------|
| [`/develop-feature`](./develop-feature) | Build a feature using docs as source of truth |
| [`/fix-issue`](./fix-issue) | Fix a bug with context from docs and code |
| [`/trace-flow`](./trace-flow) | Trace code flow with business context from docs |

## How to Use

In Claude Code, simply type the command name:

```
/develop-feature user-auth
```

Claude will read the command definition and follow its instructions.
